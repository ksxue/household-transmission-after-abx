README

These consolidated species abundance profiles from the Human Microbiome Project were processed as part of  B.H. Good, L.B. Rosenfeld. Eco-evolutionary feedbacks in the human gut microbiota. biorXiv 2022.01.26.477953.

This files were downloaded from the Github files of the same names.
https://github.com/bgoodlab/microbiome_ecoevo_correlations/tree/main

In the header of HMP1-2_ids_order.txt, I replaced a space between the "country" and "continent" column names with a tab.
